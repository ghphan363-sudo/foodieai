# Food Analyzer - Trình Phân Tích Món Ăn

## Giới thiệu

Food Analyzer - Trình Phân Tích Món Ăn là ứng dụng web sử dụng trí tuệ nhân tạo (AI) để nhận diện món ăn thông qua hình ảnh và cung cấp thông tin dinh dưỡng một cách nhanh chóng và trực quan.

Người dùng chỉ cần tải ảnh món ăn lên, hệ thống sẽ tự động phân tích và hiển thị các thông tin như:
- Lượng calo
- Chất đạm
- Chất béo
- Chất bột đường
- Đánh giá tổng thể dinh dưỡng của món ăn

Ứng dụng được thiết kế với giao diện hiện đại, thân thiện và dễ sử dụng trên cả điện thoại và máy tính.

---

## Tính năng chính

- 📷 Tải ảnh món ăn từ thiết bị

## Cách chạy dự án

```bash
# clone repo
git clone https://github.com/ItzFalco08/project.git

# install packages
cd server
npm install
cd client
npm install

# run the program
cd client; npm run dev
cd server; node index.js

# fill local.env (client), .env (server) correctly.
```
